# Imágenes de Transporte MAHE

Sube cada imagen a la carpeta que le corresponda. Nombres en minúsculas, sin espacios ni acentos (usa guiones), formato `.jpg`/`.png` (o `.webp` si puedes, pesa menos).

| Carpeta | Qué va aquí | Tamaño recomendado |
|---|---|---|
| `logo/` | Logo de Transporte MAHE (fondo transparente PNG si es posible) | logo.png, ideal 512×512 o más |
| `hero/` | Foto principal de portada (unidad/flota, o personal siendo trasladado) | 1920×1080 aprox. |
| `flota/` | Fotos de las unidades/camionetas/autobuses | 1200×800 aprox., varias fotos |
| `equipo/` | Fotos del equipo/choferes (opcional, da confianza) | 800×800 aprox. |
| `galeria/` | Cualquier otra foto de operación, rutas, clientes, etc. | libre |

Cuando subas las imágenes, avísame los nombres de archivo (o solo dime "ya subí las fotos") y yo edito el sitio para colocarlas en las secciones correspondientes (logo en el header, foto de hero en la portada, etc.).
